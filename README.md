# 🧶 Kumihimo Designer - Pattern (7x74)

**Kumihimo Designer** est un outil interactif web permettant de concevoir et de visualiser des motifs de tressage Kumihimo en temps réel. Il combine un patron 2D précis avec une simulation 3D interactive pour un rendu réaliste du bracelet final.

## 🚀 Démo en ligne
Accédez directement à l'outil ici : [https://boubalouf.github.io/kumihimopatern.io](https://boubalouf.github.io/kumihimopatern.io)

## ✨ Fonctionnalités

* **Personnalisation des 7 fils** : Changez les couleurs via un sélecteur intuitif.
* **Visualisation 3D** : Un moteur **Three.js** génère un cylindre texturé pour simuler le bracelet physique (rotation et zoom inclus).
* **Patron 2D Exact** : Affichage du diagramme de tressage pixelisé (7 colonnes sur 74 rangées).
* **Bibliothèque de Couleurs** : 
    * Ajoutez vos couleurs favorites.
    * Sauvegarde automatique dans votre navigateur (`localStorage`).
    * Suppression facile par simple clic.
* **Générateur Aléatoire** : Mélangez les fils en utilisant soit vos couleurs enregistrées, soit des couleurs aléatoires pour trouver de nouvelles inspirations.
* **Optimisé Mobiles** : Interface responsive adaptée aux smartphones et tablettes.
